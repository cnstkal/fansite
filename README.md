# novamoon fansite

가상 걸그룹 NOVAMOON 팬사이트 목업 (React + Vite)

## 로컬 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

## 빌드

```bash
npm run build
```

`dist` 폴더에 정적 파일이 생성됩니다.

## 배포 (Vercel 예시)

1. 이 폴더를 GitHub 레포에 push
2. vercel.com에서 GitHub 레포 Import
3. 자동으로 Vite 프로젝트를 감지해 배포됨
